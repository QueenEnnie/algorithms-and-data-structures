## Запуск проекта
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/QueenEnnie/algorithms-and-data-structures.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd lab0
   ```
3. **Запуск всех задач**
    ```bash
        py all_src.py
    ```

4. **Запуск всех тестов задач**
    ```bash
        pytest -v
    ```
